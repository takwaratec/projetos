Este workflow deve ser utilizado para gerar prompts de imagem (DALL-E, Midjourney, Stable Diffusion ou ferramentas internas) que sigam a identidade @takwaratec.

### Regras de Ouro para a Imagem:

1. **Branding Obrigatório**:
    - A imagem DEVE ser gerada com um **pass-partout branco** (border).
    - Inclua no prompt: "discreet signature in the bottom left corner reading 'Takwara 2026'".
2. **Estética Dominante**:
    - Para diagramas ou ilustrações não-realistas: "Sketched style, line art inspired by Henfil's fluid strokes or Andy Warhol's pop-art stencil/flat colors".
    - Para imagens técnicas: "Professional, realistic photography".
3. **Restrições Éticas (Compliance GEMINI)**:
    - **NUNCA** use azul/verde brilhante para tratamento de bambu (evite alusão a CCA/CCB).
    - Estruturas geodésicas devem usar o **Sistema Takwara** (tubos, PU, conexões de biomassa).
    - O bambu deve ter o aspecto de **Guadua angustifolia** (nós proeminentes, paredes grossas).
