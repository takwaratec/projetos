This workflow must be used exclusively for processing denunciation reports and high-impact materials for Advocacy (e.g., COP30).

### Execution Instructions:

1. **Anchor Protocol (Module D)**:
   - Convert ALL references `[^n]` to the inline standard: `[(Source, Year)](URL)`.
   - Remove the list of footnotes at the end of the document.

2. **Precise Endpoints Guarantee**:
   - Validate that links point to the final document (e.g., PDF of the Law, House Minutes), not generic pages.
   - Add "Access Date" metadata if necessary in the consolidated bibliography.

3. **Technical Rigor and Objectivity**:
   - Eliminate any qualitative tone generated by AI that is not fact-based.
   - Refer to `GEMINI.md` Module A to validate technical terms (Native Bamboo, Vegetable PU).

4. **Synchronization**:
   - Ensure structural improvements are applied simultaneously to Portuguese and Spanish versions.
