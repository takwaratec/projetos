# Management Skill

Esta Skill foca na operação técnica e administrativa do repositório, garantindo que o fluxo de trabalho digital seja eficiente e versionado.

## Operações GitHub
- **Commits**: Mensagens claras e em português (ou inglês para repositórios técnicos).
- **Tradução**: Manutenção de paridade entre arquivos PT, EN e ES.
- **Estrutura**: Respeitar rigorosamente a hierarquia de pastas definida.

## Ferramentas de Gestão
- Uso de `mkdocs.yml` para estruturação do site.
- Monitoramento de GitHub Actions para build e deploy.
- Gestão de documentos restritos e financeiros na pasta `99_RESTRITO`.

## Manutenção do Repositório
- Limpeza periódica de arquivos temporários ou redundantes.
- Verificação de links quebrados entre documentos `docs` e `Arquivo Takwara`.
