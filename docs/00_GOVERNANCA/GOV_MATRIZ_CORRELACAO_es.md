## Observaciones de Gobernanza
- **Master GitHub**: El repositorio GitHub es la fuente oficial para el versionamiento y preservación para el Zenodo.
- **Edición en GDocs**: Los cambios realizados por el GT en GDocs deben ser reportados o integrados en GitHub mensualmente o según los hitos del proyecto (ej: COP30).
- **Sin Veneno**: Cualquier cambio técnico debe pasar por el escrutinio de [`GEMINI.md`](../GEMINI.md).
