## 6. Sincronização DOI — Protocolo de Atualização

Para manter este banco atualizado, executar os seguintes comandos após cada publicação no Zenodo:

```bash
# Verificar todos os DOIs do acervo
grep -rh --include="*.md" -oE "https://doi\.org/[^ )\"']+" docs/ | sort -u

# Listar DOIs com placeholder
grep -rn --include="*.md" "zenodo.XXXXXXX" docs/

# Contar links externos únicos
grep -rh --include="*.md" -oE "https?://[a-zA-Z0-9./_?=#&%-]+" docs/ | \
  grep -v "img.shields.io" | grep -v "creativecommons.org" | sort -u | wc -l
```
