## Sobre o Projeto

A **Plataforma Amazônia Regenerativa** articula quatro frentes
complementares de bioeconomia circular na Amazônia Legal:

| Núcleo | Responsável | Foco |
| :-- | :-- | :-- |
| 🎋 **Bambu & Biorrefinaria** | Fabio Resck Takwara | Pirólise, biochar, SMGA, crédito de carbono, geodésicas, PU Vegetal |
| ⚙️ **Máquinas & Equipamentos** | Jesiel | Kits de tecnologia social, fornos, prensas |
| 🏗️ **Bioconstrução** | Filemon Tiago | HIS, adobe, sistemas híbridos |
| 👩 **Mulheres & Gestão** | Núcleo feminino | Cooperativas, artesanato, governança comunitária |
